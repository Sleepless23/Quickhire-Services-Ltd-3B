## Quick start

### Prerequisites
- Python 3.10+ recommended
- git

### Setup
```bash
# clone
git clone https://github.com/Sleepless23/Quickhire-Services-Ltd-3B.git

cd Quickhire-Services-Ltd-3B

# create venv (recommended)
python -m venv .venv
# activate (Windows)
.venv\Scripts\activate
# mac / linux
source .venv/bin/activate

# install deps
pip install -r requirements.txt

# open another terminal and run app on cli
python main.py
```
